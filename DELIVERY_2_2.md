# ISAP Delivery 2.2 — Presentation Viewer

## Included
- Open presentations directly from card or list view
- Full-screen viewer with slide rail, main stage and information panel
- Previous/next navigation and keyboard arrow navigation
- Zoom from 50% to 150%
- Speaker notes per generated slide
- Linked products, industries, audiences and assets
- Favorite toggle inside the viewer
- Persistent last slide, zoom and panel state via localStorage
- Responsive tablet and mobile layout

## Current scope
The viewer creates structured preview slides from presentation metadata. Native PowerPoint rendering and real slide extraction are planned for Slide Intelligence.
