# ISAP Delivery 2.1.1 - Stabilization

This hotfix stabilizes Delivery 2.1 without changing the Presentation Library feature set.

## Fixes

- Administration opens correctly again.
- Presentation count is derived safely from the provided count or administration state.
- CRM Workspace tolerates meetings without CRM data.
- Missing CRM arrays render empty states instead of crashing the view.
- Missing meeting and forecast values are handled safely.
