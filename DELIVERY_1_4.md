# Delivery 1.4 - Document Intelligence

## Included

- Automatic analysis pipeline for ready assets
- Analysis states: queued, extracting, analysing, summarising and ready
- Executive summary generation
- Keyword, topic, industry and product detection
- Language detection
- Page or slide count estimation
- Confidence, execution and provider metadata
- Editable analysis results
- Manual reanalysis
- Search across detected intelligence metadata

## Technical note

This delivery uses the local ISAP heuristic engine and does not send documents to an external AI provider. The provider-ready data model can be connected to OpenAI or Azure OpenAI in a later integration delivery.
